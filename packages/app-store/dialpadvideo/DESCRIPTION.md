# Dialpad Meetings

Create Dialpad meeting links for Cal.com bookings.

## Setup

- Set `DIALPAD_API_KEY` in your server environment (do **not** expose it to the client).

## Notes

- The meeting link is created when a booking is created.
- This integration uses the Dialpad API v2. See [Dialpad Developer Docs](https://developers.dialpad.com/docs/welcome).
